# 24/7 Star Parking

Sistema inteligente de gestión de parqueadero desarrollado como proyecto ADSO SENA.

## Tecnologías utilizadas
- Java
- HTML5
- CSS3
- JavaScript
- MySQL

## Funcionalidades
- Registro de vehículos
- Gestión de usuarios
- Control de espacios
- Reportes
- Pagos

## Arquitectura
El sistema fue desarrollado bajo la arquitectura MVC (Modelo-Vista-Controlador).

## Estructura del Proyecto
- **DOCUMENTACION**: Evidencias e informes técnicos.
- **CODIGO_FUENTE**: Backend, Frontend y DATABASE del sistema.
